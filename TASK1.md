### 1. Некорректные результаты поиска по ключевому слову ```Аналитик```

- Описание: При вводе в поисковую строку запроса ```Аналитик``` среди результатов отображаются вакансии, не соответствующие запросу, например, ```Руководитель Data Science в команду монетизации``` и ```Руководитель группы продаж по работе с агентствами```.
- Приоритет: ```High```
- Пояснение: Критическая ошибка функциональности поиска. Пользователь ожидает увидеть вакансии, связанные с аналитикой, а получает невалидные результаты.

### 2. Некорректная фильтрация по специализации ```Поддержка пользователей```

- Описание: При активации фильтра ```Поддержка пользователей``` отображаются вакансии по всем специализациям.
- Приоритет: ```High```
- Пояснение: Основная функция фильтрации не работает. Пользователь не может найти результаты по нужной специализации.

![2](/img/h2.png?raw=true)

### 3. Некорректная фильтрация по типу работы ```Можно удаленно```

- Описание: При активации фильтра ```Можно удаленно``` отображаются вакансии с форматом работы ```офис```, ```офис и удаленно```, а не только ```удаленно```.
- Приоритет: ```High```
- Пояснение: Фильтр работает некорректно. Пользователь, ищущий только удаленные вакансии, получает невалидные результаты.

![3](/img/h4.png?raw=true)

### 4. Некорректная фильтрация по городу

- Приоритет: ```High```
- Пояснение: Фильтр по локации не работает. Пользователь, ищущий работу в определенном городе, получает результаты из других регионов.
